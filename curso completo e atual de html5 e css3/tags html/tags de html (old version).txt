------------NEGRITO------------



<b> para abrir a tag para escrever em negrito

exemplo: <p>Nessa frase temos um <b>termo em negrito usando a tag B</b></p>

resultado: Nessa frase temos um *termo em negrito usando a tag B* #bloco de notas não possui formatação em negrito, logo, * para representar.


tag B é (não semantica)



------------STRONG------------



<strong> para abrir a tag para escrever em strong

exemplo: <p>Nessa frase temos um <strong>termo em destaque usando a tag STRONG</strong></p>

resultado: Nessa frase temos um *termo em destaque usando a tag STRONG* #bloco de notas não possui formatação em STRONG, logo, * para representar.

faz a mesma coisa que o NEGRITO, mas com um significado diferente e com SEMANTICA, é usado para dar um sentido para a frase: "TERMO FORTE, TERMO DE DESTAQUE".



------------ITÁLICO------------



<i> para abrir a tag para escrever em itálico

exemplo: <p>Nessa frase temos um <i>termo em destaque usando a tag I</i></p>

resultado: Nessa frase temos um *termo em itálico usando a tag I.



------------ENFÂSE------------



<em> para abrir a tag para escrever em enfase

exemplo: <p>Nessa frase temos um <em>termo em destaque usando a tag I</em></p>

resultado: Nessa frase temos um *termo em destaque usando a tag I.

faz a mesma coisa que o ITÁLICO, mas com um significado diferente e com SEMANTICA, é usado para dar um sentido para a frase: "dar ênfase na frase".

<audio> para abrir a tag para criar uma tag de áudio.

        <source src="midia/guanacast-33.mp3" type="audio/mpeg"> <!--para mp3-->
        <source src="midia/guanacast-33.ogg" type="audio/ogg">  <!--para ogg-->

        ></audio>
    </audio>



========preload======== 

  none → não carrega nada
  metadata → carrega só informações do áudio (duração, etc.)
  auto → pode carregar o arquivo inteiro



========Autoplay========

Faz o áudio começar automaticamente quando a página abre.

⚠️ Observação importante:
A maioria dos navegadores bloqueia autoplay com som.
Normalmente só funciona se o áudio estiver mutado.


========Controls========

Mostra os controles do player para o usuário.

Exemplo de controles:

▶️ play

⏸ pause

🔊 volume

barra de tempo

Sem controls, o áudio toca mas o usuário não vê nada.


======== loop========

Faz o áudio recomeçar automaticamente quando termina.

Exemplo:

🎵 música acaba
↩️ volta para o início
🎵 toca novamente
