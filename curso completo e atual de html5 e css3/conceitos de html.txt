--------COMO A INTERNET FUNCIONA--------


Cliente: você abre o Chrome, digita um site, isso é o cliente fazendo um pedido.

Modem: transforma o sinal da operadora (fibra, cabo, etc.) em dados que seus dispositivos entendem e faz o caminho inverso também.

Internet: É a rede gigante que conecta vários computadores (servidores) no mundo.

Servidor: Um computador que guarda sites, sistemas, dados (servidor do Googlem servidor do Instagram, servidor de um site qualquer)

IP: endereço/CPF do servidor

DNS: agenda que encontra o servidor, recebe o nome do site (google.com), procura qual IP pertence a esse nome e devolve o IP pro navegador.

 Como tudo funciona junto (bem rápido)

1️⃣ Você (cliente) digita google.com
2️⃣ O navegador pergunta ao DNS:
“qual o IP desse site?”
3️⃣ O DNS responde com o IP
4️⃣ Seu computador se conecta ao servidor pelo IP
5️⃣ O servidor envia o site
6️⃣ Você vê a página 🎉

--------DIFERENTES FORMATAÇÕES DE IMAGEM--------


- JPEG:  alta compactação, menor qualidade;
- PNG:  média compactação, maior qualidade, permite transparência;
- GIF:  baixa qualidade, permite transparência e animação (menos usado)

Transparencia: arquivos sem fundo



Compactação é o processo de diminuir o tamanho do arquivo da imagem, pra ela:


-ocupar menos espaço 💾
-carregar mais rápido na internet 🚀
-ser mais fácil de enviar (WhatsApp, site, email)
ou seja:  mesma imagem “visualmente parecida”, porém mais leve.



RESUMO RÁPIDO


Compactação =  deixar a imagem mais leve

Pode perder qualidade ou não

JPEG  perde um pouco → fica bem menor

PNG não perde →  fica maior
