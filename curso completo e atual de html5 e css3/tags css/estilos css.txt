------------ ESTILOS EM CSS INTERNO/LOCAL ---------



É MELHOR QUE INLINE, MAS É PIOR QUE CSS EXTERNO, ESSA É MELHOR QUANDO VOCE VAI MODIFICAR SÓ UM DOCUMENTO, SÓ UMA PAGINA



exemplo:

    <title>Estilos internos/locais</title>
    <!--o parametro que estiver dentro de style será alterado-->
    <style>
        body {
            background-color: lightsteelblue;  ========================================================================================================================
            font-family: Arial, Helvetica, sans-serif; ================================================================================================================
            font-size: 20px; ==========================================================================================================================================
        }   
        h1 {
            color:darkblue;
            background-color:lightblue;
        }
        h2 {
            color:darkred;
        }
        p {
            text-align: justify;
        }
    </style>



------------ ESTILOS EM CSS EXTERNO (DOCUMENTO ESPECIFICO PARA CSS) ---------



É OTIMO PARA GUARDAR ESTILOS E USAR QUANDO QUISER, ALTERA EM TODOS AS PAGINAS, NÃO PRECISA ALTERAR UMA PAGINA POR VEZ



TAG USADA NO  CSS EXTERNO:



 <link rel="stylesheet" href="style.css">!--tag para definir o caminho do ARQUIVO CSS-->



exemplo:


    <title>Segunda página</title>
    <link rel="stylesheet" href="style.css">!--tag para definir o caminho do ARQUIVO CSS-->
</head>
<body>
    <h1>Segunda página</h1>
    <h2>Teste de estilos</h2>
    <p>Lorem ipsum, dolor sit amet consectetur adipisicing elit. Ducimus soluta corporis aperiam quaerat odio totam quae, aspernatur incidunt saepe nobis quibusdam, recusandae excepturi deserunt eveniet! At animi non quibusdam ullam.</p>
    <p><a href="index.html">Voltar</a></p>
</body>
</html>



--------- PODE USAR CSS INTERNO, EXTERNO E INLINE JUNTOS? ---------




SIM PODE




ordem de prevalecimento (NÃO É ORDEM DE MELHOR)

1:Inline (coisas muito especificas, alterar uma coisa MUITO MAS MUITO ESPECIFICA, USE SÓ PARA ISSO)
2:Local/ interno (bom para quando é só uma página, mas se aumentar é so copiar e fazer um documento css)
3:Externo (foda dms)
