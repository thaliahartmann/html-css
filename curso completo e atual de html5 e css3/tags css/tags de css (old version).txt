------------ TAG <SPAN> ---------


É UMA TAG HTML INLINE, SERVE PARA MARCAR UMA PEQUENA PARTE DO TEXTO, NORMALMENTE UMA PALAVRA OU TRECHO, PARA DEPOIS ESTILIZAR COM CSS.



exemplo:

<p>Eu gosto de <span>Android</span>.</p>



resultado:

A palavra Android fica separada do restante do texto para receber estilo.



EXEMPLO COM CLASS:

<p>Eu gosto de <span class="destaque">Android</span>.</p>

<style>
    .destaque {
        color: green;
        font-weight: bold;
    }
</style>



resultado:

A palavra Android ficará verde e em negrito.

✔ destacar palavra
✔ colorir parte do texto
✔ aplicar class em trecho pequeno
✔ aplicar id em palavra específica

--------- MEDIDAS (TAMANHOS DA LETRA) EM CSS ---------



📏 MEDIDAS ABSOLUTAS
(são fixas, não mudam dependendo da tela)

px (pixels)
→ unidade mais comum em telas
→ representa pontos na tela
→ MUITO usada para tamanhos

cm (centímetros)
→ unidade real de medida física (pouco usada em telas)

mm (milímetros)
→ menor que cm, também baseada em medida real

in (polegada)
→ 1 polegada = 2.54 cm

pt (points)
→ usado mais em impressão (1pt ≈ 1/72 de polegada)

pc (picas)
→ usado em tipografia (1pc = 12pt)

⚠️ Observação:
Na prática, o mais usado é:
→ px



📐 MEDIDAS RELATIVAS
(se adaptam ao tamanho da tela ou elemento)

em
→ relativo ao tamanho da fonte do elemento pai
→ Ex: se o pai tem 16px, 1em = 16px

rem (root em)
→ relativo ao tamanho da fonte do elemento raiz (html)
→ mais previsível que em

ex
→ relativo à altura da letra "x" (pouco usado)

vw (viewport width)
→ relativo à largura da tela
→ 1vw = 1% da largura da tela

vh (viewport height)
→ relativo à altura da tela
→ 1vh = 1% da altura da tela

%
→ relativo ao tamanho do elemento pai

---

📌 MEDIDAS MAIS RECOMENDADAS PARA FONT-SIZE

px
→ simples e direto
→ muito usado para iniciantes

em
→ mais flexível
→ bom para layouts adaptáveis

rem
→ melhor controle geral
→ muito usado profissionalmente



🧠 RESUMO

Medidas absolutas → tamanho fixo
Medidas relativas → tamanho adaptável

px → padrão simples
em/rem → padrão moderno e responsivo


QUEM É PAI?

Elemento pai = elemento que envolve outro elemento (filho)

Exemplo:

<div>
    <p>Texto</p>
</div>

div → pai
p → filho

---

IMPORTÂNCIA:

Algumas medidas dependem do pai:

em → usa o tamanho do pai
% → usa o tamanho do pai

rem → NÃO usa o pai (usa o html)

---

RESUMO:

pai = elemento de fora
filho = elemento de dentro

--------- 📦 MODELOS DE CAIXA ---------



DICA DE RUBI:

-quando for alterar o tamanho, borda e tals, da para alterar
no inspecionar da pagina no navegador, nao altera no arquivo original
mt foda

Parâmetros:


height  -> altura
width   -> largura
border  -> borda
padding -> espaço interno (entre conteúdo e borda)
margin  -> espaço externo (fora da borda)
outline -> contorno externo (não ocupa espaço)


Tipos de caixas:

🔷 Block-level:

- Começa em nova linha
- Ocupa toda a largura do elemento pai
- Ex: <div>, <h1>, <p>, <main>, <section>, <article>, <header>, <footer>

🔶 Inline-level:

- Não quebra linha
- Ocupa apenas o tamanho do conteúdo
- Ex: <span>, <a>, <strong>, <em>, <small>

🟡 Inline-block (extra):

- Fica na mesma linha
- Aceita width e height
- Ex: <button>, <input>, <select>



--------- PROPRIEDADES DE BORDA (border) ---------



largura  -> border-width: 10px;

estilo   -> border-style:
             solid   (linha sólida)
             dashed  (tracejada)
             dotted  (pontilhada)
             double  (dupla)
             groove  (efeito 3D)

cor      -> border-color: red;


🔹 SHORTHAND:
border: 10px solid red;   (largura | estilo | cor)

--------- GROUPING TAGS (TAGS DE AGRUPAMENTO)  ---------



🔹 O QUE SÃO:

Servem para organizar e estruturar o conteúdo da página
(dividem o site em partes lógicas)

inves de so colocar tudo em baixo do outro, voce organiza com as TAGS DE AGRUPAMETO



📌 RESUMO RÁPIDO

div → genérico (sem significado)
header → topo
nav → navegação
main → conteúdo principal
section → divisão por tema
article → conteúdo independente
aside → lateral
footer → rodapé



============================================
ENTENDIMENTO RÁPIDO:

DIV = caixa sem etiqueta 📦
header/nav/main = caixas com etiqueta 🏷️
=========================================


📦 TAG GENÉRICA

<div> → contêiner genérico (sem significado semântico)

ex:

<div>
    <p>Texto</p>
</div>

resultado:

apenas agrupa elementos (usado para layout/estilo)



📦 TAGS SEMÂNTICAS (com significado)

🔹 <header> → topo

usado para:

- título
- logo
- navegação

ex:

<header>
    <h1>Meu site</h1>
</header>

resultado:

define o cabeçalho da página/seção



🔹 <nav> → navegação

ex:

<nav>
    <a href="#">Home</a>
    <a href="#">Contato</a>
</nav>

resultado:

área de links do site



🔹 <main> → conteúdo principal

ex:

<main>
    <p>Conteúdo principal</p>
</main>

resultado:

parte mais importante da página (1 por página)



🔹 <section> → seção temática

ex:

<section>
    <h2>Notícias</h2>
</section>

resultado:

divide o conteúdo em partes com tema



🔹 <article> → conteúdo independente

ex:

<article>
    <h2>Post</h2>
    <p>Texto...</p>
</article>

resultado:

conteúdo que pode existir sozinho (ex: post, notícia)



🔹 <aside> → conteúdo lateral

ex:
<aside>
    <p>Publicidade</p>
</aside>

resultado:

conteúdo secundário (relacionado, mas não principal)



🔹 <footer> → rodapé

ex:
<footer>
    <p>© 2026</p>
</footer>

resultado:

parte final da página



📦 TAGS DE FORMULÁRIO


🔹 <fieldset> → agrupa campos

ex:
<fieldset>
    <legend>Dados</legend>
</fieldset>

resultado:

organiza inputs com um título



🔹 <optgroup> → agrupa opções

ex:
<select>
    <optgroup label="Frutas">
        <option>Maçã</option>
    </optgroup>
</select>

resultado:

agrupa opções dentro do select

--------- BOX SHADOW ---------




- Cria sombra ao redor de um elemento (caixa)

- A melhor forma de configurar é pelo inspecionar



🔹 INSPECIONANDO:

- box-shadow: ☐ 1px 1px 1px black;
- tem que clicar nesse quadradinho para abrir as configurações de sombra


🔹 SINTAXE:

box-shadow: deslocamento-x deslocamento-y desfoque espalhamento cor;


🔹 EXEMPLO BÁSICO

box-shadow: 5px 5px 10px black;

resultado:
sombra deslocada para direita (5px) e para baixo (5px), com leve desfoque



🔹 EXPLICANDO CADA PARTE

box-shadow: 5px 5px 10px 0px black;
             |   |   |    |   |
             x   y  blur spread cor

x → move para direita (negativo = esquerda)
y → move para baixo (negativo = cima)
blur → desfoque (quanto maior, mais suave)
spread → aumenta ou diminui o tamanho da sombra
cor → cor da sombra



🔹 EXEMPLO COMPLETO

box-shadow: 5px 5px 15px 5px rgba(0, 0, 0, 0.5);

resultado:

sombra mais suave, maior e com transparência



🔹 SOMBRA INTERNA

box-shadow: inset 5px 5px 10px black;

resultado:

sombra fica DENTRO da caixa



🔹 SEM DESFOQUE (sombra dura)

box-shadow: 5px 5px 0px black;

resultado:

sombra sólida, sem suavização



🔹 VÁRIAS SOMBRAS

box-shadow: 5px 5px 10px black, -5px -5px 10px gray;

resultado:

duas sombras (uma em cada direção)



--------- BORDER-RADIUS(BORDAS ARREDONDADAS) ---------



🔹 O QUE É:

Arredonda os cantos de uma caixa



🔹 EXEMPLO BÁSICO

border-radius: 10px;

resultado:
todos os cantos ficam arredondados igualmente



🔹 ARREDONDANDO CANTOS ESPECÍFICOS

border-top-left-radius: 10px;
border-top-right-radius: 20px;
border-bottom-right-radius: 30px;
border-bottom-left-radius: 40px;

resultado:

cada canto com um arredondamento diferente



🔹 SHORTHAND (forma resumida)

border-radius: 10px 20px 30px 40px;

ordem:

top-left → top-right → bottom-right → bottom-left

resultado:

cada canto recebe um valor diferente



🔹 OUTRAS FORMAS DE SHORTHAND

border-radius: 10px;
resultado:
todos iguais

border-radius: 10px 20px;
resultado:
top-left + bottom-right = 10px
top-right + bottom-left = 20px

border-radius: 10px 20px 30px;
resultado:
top-left = 10px
top-right + bottom-left = 20px
bottom-right = 30px



🔹 FAZER UM CÍRCULO

border-radius: 50%;

⚠️ precisa que largura = altura

ex:
width: 100px;
height: 100px;

resultado:
vira um círculo perfeito



🔹 FAZER UM BOTÃO ARREDONDADO

border-radius: 20px;

resultado:
bordas suaves (muito usado em botões)



🔹 DICA

valores maiores → mais arredondado
valores menores → menos arredondado
