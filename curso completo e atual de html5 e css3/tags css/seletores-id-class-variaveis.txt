- ai voce nomeia

html:

<h1 class="destaque">Aprendendo HTML</h1>

class ="" o nome é da sua escolha

- não deve botar h1 antes de ".destaque" ou qualquer outra tag

--------- VARIÁVEIS EM CSS ---------


Variáveis CSS guardam valores que você usa várias vezes no
projeto. Se precisar mudar uma cor ou fonte, muda só em um
lugar e atualiza tudo automaticamente.


--------- COMO DECLARAR ---------


As variáveis são declaradas dentro de :root
:root é o elemento raiz da página (equivale ao html)
Toda variável começa com -- (dois traços)

    :root {
        --nome-da-variavel: valor;
    }

Exemplos:
    :root {
        --cor-primaria: #3498db;
        --cor-fundo: #f0f0f0;
        --fonte-padrao: Arial, sans-serif;
        --tamanho-fonte: 16px;
        --espacamento: 10px;
    }


---------- COMO USAR ----------


Para usar a variável, chame com var(--nome-da-variavel)

    h1 {
        color: var(--cor-primaria);
        font-family: var(--fonte-padrao);
        font-size: var(--tamanho-fonte);
    }


---------- POR QUE USAR VARIÁVEIS? ----------


Sem variável:
    h1    { color: #3498db; }
    nav   { background-color: #3498db; }
    footer { border-color: #3498db; }

    Se precisar mudar a cor → tem que trocar em 3 lugares.

Com variável:
    :root { --cor-primaria: #3498db; }

    h1    { color: var(--cor-primaria); }
    nav   { background-color: var(--cor-primaria); }
    footer { border-color: var(--cor-primaria); }

    Se precisar mudar a cor → muda só no :root.

<!DOCTYPE html>
<html lang="pt-br">
<head>
    <meta charset="UTF-8">
    <title>Exemplo</title>
    <style>

        /* VARIÁVEIS GLOBAIS */
        :root {
            --cor-primaria: #3498db;
            --cor-fundo: #f0f0f0;
            --cor-destaque: lightyellow;
            --fonte-padrao: Arial, sans-serif;
            --espacamento: 10px;
        }

        /* TAG */
        body {
            font-family: var(--fonte-padrao);
            background-color: var(--cor-fundo);
        }

        /* ID */
        #topo {
            background-color: var(--cor-primaria);
            color: white;
            padding: var(--espacamento);
        }

        /* CLASS */
        .menu a {
            color: white;
            margin-right: var(--espacamento);
            text-decoration: none;
        }

        /* CLASS */
        .noticia {
            border: 1px solid var(--cor-primaria);
            padding: var(--espacamento);
            margin-bottom: var(--espacamento);
        }

        /* ID */
        #principal {
            background-color: var(--cor-destaque);
            padding: var(--espacamento);
        }

        /* TAG */
        footer {
            background-color: var(--cor-primaria);
            color: white;
            padding: var(--espacamento);
        }

    </style>
</head>
